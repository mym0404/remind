# Responsive Card Grid

## 문제

카드 목록이 viewport 너비에 맞춰 열 수를 바꾸게 만든다.

## 핵심 개념

- CSS Grid를 쓴다.
- auto-fit 또는 auto-fill과 minmax를 쓴다.
- 카드 최소 너비를 고정하고 남는 공간을 분배한다.

## 풀이 흐름

container에 grid를 적용하고 column 정의를 `repeat(auto-fit, minmax(min(100%, 220px), 1fr))`처럼 둔다. 이렇게 하면 좁은 화면에서는 한 열, 넓은 화면에서는 가능한 만큼 열이 늘어난다.

## 핵심 코드

```css
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(min(100%, 220px), 1fr));
  gap: 12px;
}
```

## 테스트가 보는 것

- 카드들이 grid container 안에 렌더링되는지 본다.
- CSS에 intrinsic responsive grid 규칙이 있는지 확인한다.

## 실수 포인트

- 고정 3열 grid는 모바일에서 overflow를 만든다.
- 카드 폭만 정하고 grid column을 안 바꾸면 남는 공간 분배가 어색해진다.
