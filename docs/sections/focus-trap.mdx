# Focus Trap

## 언제 쓰나

Focus Trap은 modal처럼 열린 영역을 먼저 처리해야 할 때 쓴다. Tab focus를 영역 안에서만 순환시킨다.

열릴 때는 닫기 버튼이나 첫 입력값으로 focus를 옮기고, 닫힐 때는 이전 focus로 되돌린다.

:::warning
Focus Trap은 focus 이동만 다룬다. 배경 `inert`, scroll lock, Escape 닫기, portal은 dialog 쪽 책임이다.
:::

## 바로 쓰는 코드

```tsx title="useFocusTrap.tsx"
const focusableSelector = [
  "a[href]",
  "button",
  "input",
  "[contenteditable]:not([contenteditable='false'])",
  "[tabindex]",
  // ...
].join(",");

const isVisible = (element) => {
  const style = window.getComputedStyle(element);

  return (
    style.display !== "none" &&
    style.visibility !== "hidden" &&
    element.getClientRects().length > 0
  );
};

const isFocusable = (element) =>
  // Selector only finds candidates. Filter the real Tab order again.
  element.matches(focusableSelector) &&
  !element.closest("[hidden], [inert], [aria-hidden='true']") &&
  !element.matches(":disabled") &&
  isVisible(element);

const getTabbableElements = (container) =>
  [...container.querySelectorAll<HTMLElement>(focusableSelector)].filter(
    (element) => isFocusable(element) && element.tabIndex >= 0,
  );

const getActiveElement = () => {
  const element = document.activeElement;

  if (!(element instanceof HTMLElement) || element === document.body) {
    return undefined;
  }

  return element;
};

const useFocusTrap = ({
  isActive,
  containerRef,
  initialFocusRef,
}) => {
  const previousFocusRef = useRef(undefined);

  useEffect(() => {
    if (!isActive) {
      return;
    }

    const container = containerRef.current;

    if (!container) {
      return;
    }

    previousFocusRef.current = getActiveElement(); // [!code highlight]

    const tabbableElements = getTabbableElements(container);
    const initialFocusElement =
      initialFocusRef?.current && isFocusable(initialFocusRef.current)
        ? initialFocusRef.current
        : tabbableElements[0] ?? container;

    initialFocusElement.focus(); // [!code highlight]

    const handleKeyDown = (event) => {
      if (event.key !== "Tab") {
        return;
      }

      const currentTabbableElements = getTabbableElements(container); // [!code highlight]

      if (currentTabbableElements.length === 0) {
        event.preventDefault();
        container.focus(); // Requires tabIndex={-1} on the container.
        return;
      }

      const firstElement = currentTabbableElements[0];
      const lastElement =
        currentTabbableElements[currentTabbableElements.length - 1];
      const activeElement = getActiveElement();
      const isFocusOutside =
        !activeElement || !container.contains(activeElement);

      if (event.shiftKey) {
        if (activeElement === firstElement || isFocusOutside) {
          event.preventDefault();
          lastElement.focus(); // [!code highlight]
        }

        return;
      }

      if (activeElement === lastElement || isFocusOutside) {
        event.preventDefault();
        firstElement.focus(); // [!code highlight]
      }
    };

    document.addEventListener("keydown", handleKeyDown);

    return () => {
      document.removeEventListener("keydown", handleKeyDown);

      if (previousFocusRef.current?.isConnected) {
        // The opener may have unmounted while the trap was active.
        previousFocusRef.current.focus(); // [!code highlight]
      }
    };
  }, [containerRef, initialFocusRef, isActive]);
};
```

```tsx title="FocusTrapDialog.tsx"
const FocusTrapExample = () => {
  const [isOpen, setIsOpen] = useState(false);
  const dialogRef = useRef(null);
  const closeButtonRef = useRef(null);

  useFocusTrap({
    isActive: isOpen,
    containerRef, // [!code highlight]
    initialFocusRef: closeButtonRef,
  });

  return (
    <>
      <button type="button" onClick={() => setIsOpen(true)}>
        배송지 선택
      </button>

      {isOpen ? (
        <div className="backdrop">
          <section
            ref={dialogRef}
            aria-labelledby="delivery-title"
            aria-modal="true"
            className="dialog"
            role="dialog"
            tabIndex={-1} // [!code highlight]
          >
            <h2 id="delivery-title">배송지 선택</h2>

            {/* ... */}

            <button
              ref={closeButtonRef}
              type="button"
              onClick={() => setIsOpen(false)}
            >
              닫기
            </button>
          </section>
        </div>
      ) : undefined}
    </>
  );
};
```

## 실수 포인트

- tabbable element는 Tab을 누를 때마다 다시 계산한다.
- focus가 trap 밖에 있어도 첫 요소와 마지막 요소 경계에서 다시 안으로 돌린다.
- focus할 내부 요소가 없으면 container가 fallback이 된다.
