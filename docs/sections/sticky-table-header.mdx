# Sticky Table Header

## 문제

스크롤되는 표에서 header만 상단에 붙게 만든다.

## 핵심 개념

- overflow는 table wrapper가 가진다.
- header cell은 `position: sticky; top: 0`이다.
- background와 z-index가 겹침을 막는다.

## 풀이 흐름

의미 있는 table 구조는 유지한다. sticky 스타일은 `th`에 주고 scroll 높이는 wrapper에서 제한한다.

## 핵심 코드

```css
.table-shell {
  max-height: 240px;
  overflow: auto;
}

th {
  position: sticky;
  top: 0;
  background: #ffffff;
}
```

## 테스트가 보는 것

- table이 scroll shell 안에 들어갔는지 본다.
- CSS에 overflow와 sticky header 규칙이 함께 있는지 확인한다.

## 실수 포인트

- overflow를 body에 기대면 panel 안 sticky가 원하는 위치에 붙지 않는다.
- header 배경이 없으면 스크롤된 row가 뒤에서 비친다.
