# Sticky Table Header

## 문제

스크롤되는 표에서 header만 상단에 붙게 만든다.

## 핵심 개념

- table을 감싸는 scroll container를 만든다.
- thead th에 position: sticky와 top을 준다.
- sticky header 배경을 지정한다.

## 풀이 흐름

table 바깥에 높이가 제한된 wrapper를 두고 그 wrapper에 `overflow: auto`를 준다. header cell에는 `position: sticky`, `top: 0`, 배경색을 지정한다. sticky 기준은 가장 가까운 scroll container다.

## 핵심 코드

```css
.table-shell {
  max-height: 240px;
  overflow: auto;
}

th {
  position: sticky;
  top: 0;
  background: #ffffff;
}
```

## 테스트가 보는 것

- table이 scroll shell 안에 들어갔는지 본다.
- CSS에 overflow와 sticky header 규칙이 함께 있는지 확인한다.

## 실수 포인트

- overflow를 body에 기대면 panel 안 sticky가 원하는 위치에 붙지 않는다.
- header 배경이 없으면 스크롤된 row가 뒤에서 비친다.
