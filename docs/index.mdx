# Practice 해답집

이 문서는 practice를 풀고 난 뒤 확인하는 해답 노트다. React 개념을 따로 길게 정리하지 않고, 각 문제에서 필요한 판단과 핵심 코드만 남겼다.

문제를 먼저 풀고 막히는 지점에서 해당 페이지를 보자. 해답은 전체 완성본이 아니라 상태 설계, 이벤트 처리, 접근성 연결, 테스트가 보는 핵심만 보여준다.
